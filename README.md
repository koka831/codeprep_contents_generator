# codeprep template generator

## installation
```
git clone

export `pwd`:$PATH
export CODEPREP_TEMPLATE_PATH="`pwd`"
```

## usage
- type `codeprep_gen` where being `README.md`, then generates chapter*.md.


## settings
- `./src/*.md` : template files
- generator.js
  - input|output_*_delimiter : README|chapter*.md delimiter
